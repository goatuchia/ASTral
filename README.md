## What is ASTral?

ASTral helps you find and understand code quickly. It organizes code in a clear way so you can search important parts easily. This tool lowers the cost of reading code by up to 99%, making it faster and cheaper to analyze large codebases.

ASTral works with many programming languages and uses special tools to break down code into parts you can explore. You do not need to know coding to use it.

## 🌟 Using ASTral for Code Search

ASTral lets you search code more precisely than simple text search. 

### How to start a search:

1. Open ASTral and choose your project folder.
2. The program will organize all code files automatically.
3. Use the search box to type the name of a function, class, or other code parts you want.
4. ASTral shows you exact matches, not just nearby words.
5. Click results to see details and where the code appears.
